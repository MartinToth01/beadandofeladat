# Név Link Leírás
1 W3Schools w3schools.com Ezen az oldalon mintakódokat találhatunk akár HTML, akár CSS-hez, amik segítséget biztosítanak egyes kódok megtalálásához, illetve az oldalunk megtervezéséhez.
2 HTML validátor validator.w3.org Ezen az oldalon a HTML kódunkat ellenőrizhetjük, hogy nem gépeltünk el-e valamit, vagy esetleg a feladatban kért parancs helyett egy másikat használtunk vagy rosszabb esetben nem működik az adott megoldás.
3 CSS validátor jigsaw.w3.org/css-validator/ A CSS kódunk validálásához ezt az oldalt használjuk, hogy megbizonyosodjunk róla, hogy a CSS kódunk tökéletes, és nincs benne hiba, ami az oldal szerkezetét tönkre tenné.

Ezek az oldalak valóban hasznosnak bizonyulnak, hisz ezeken az oldalakon szerezhetünk legális segítséget a programkódunk írásában, és a kódok ellenőrzésében. A táblázatok közül ez még a legegyszerűbb és átláthatóbb, de ennél vannak bonyolultabbak is. A bootstrap verzióból a legfrissebbet használjuk a Bootstrap 5-öt.
